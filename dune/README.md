# Dune Queries

SQL queries for PulseChain analytics on [Dune Analytics](https://dune.com).

Since PulseChain is not natively indexed on Dune, these queries focus on **Ethereum-side** data (bridge contracts, sacrifice events).

## Queries

| File | Description | Dune Link |
|------|-------------|-----------|
| `bridge_flows.sql` | Bridge inflows/outflows (ETH-side) | Coming soon |

## How to use

1. Copy the `.sql` file content
2. Go to [dune.com](https://dune.com) and create a new query
3. Paste and run
4. Fork and customize as needed
