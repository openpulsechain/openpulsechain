# Frontend

React dashboard for PulseChain Analytics.

## Stack

- React + TypeScript
- Vite
- TailwindCSS
- Deployed on Vercel (free tier)

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
